# TheMightyName# 🛡️ THE MIGHTY NAME // APPAREL WITH PURPOSE

> **"A Name Above Every Name."** > *Philippians 2:9*

The Mighty Name is a premium streetwear brand built on the belief that Jesus has the power to change lives. Our mission is to share our faith through high-quality apparel while directly impacting communities in need.

---

## 🚀 The Mission
Our brand operates on a **10% Pledge**: 10% of all gross profit is diverted to social causes. We believe that apparel is a silent witness—starting with the wearer, reaching the observer, and ultimately providing aid to the marginalized.

## ✨ Key Features
- **Live Storefront**: A clean, high-contrast shopping experience.
- **Impact Tracker**: Real-time calculation of the 10% pledge based on sales.
- **The Mighty 10**: A built-in loyalty program rewarding community members after their 10th purchase.
- **Hidden Admin Command**: A secure dashboard for inventory management and revenue tracking.
- **Persistence**: Integrated `localStorage` to keep your cart and inventory safe across sessions.

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Netlify / Vercel

## 📂 Project Structure
- `/src/components/Storefront.js`: The public shop interface.
- `/src/components/Dashboard.js`: The hidden management panel (`/admin`).
- `/src/components/Impact.js`: The mission statement and live donation tracker.
- `/src/components/Info.js`: The Brand Manifesto and "Mighty 10" rewards.
